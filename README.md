\## Описание

Проект написан на Python с использованием фреймворка \*\*pytest\*\* и паттерна \*\*Page Object\*\*. Тесты взаимодействуют с веб-интерфейсом через \*\*Selenium WebDriver\*\*.



\## Структура проекта

\*   `tests/` – директория, содержащая все файлы с тестами.

\*   `pages/` – page objects, описывающие элементы и действия на веб-страницах.

\*   `conftest.py` – файл с фикстурами pytest (например, для инициализации драйвера).

\*   `set\_env.ps1` – скрипт PowerShell для установки переменных окружения, необходимых для отправки результатов в Test IT. \*\*Важно:\*\* Не забудьте отредактировать этот файл, указав свои реальные токены и идентификаторы.

\*   `.gitignore` – список файлов и папок, исключённых из версионирования Git.

\*   `README.md` – этот файл с описанием проекта.

\## Запуск тестов

Перед запуском убедитесь, что виртуальное окружение активировано.



\*\*Запустить все тесты проекта:\*\*

```bash

pytest --testit -v

```



\*\*Запустить тесты из конкретного файла:\*\*

```bash

pytest tests/test\_имя\_файла.py --testit -v

```



\*\*Запустить конкретный тест:\*\*

```bash

pytest tests/test\_имя\_файла.py::ИмяКласса::имя\_метода --testit -v

```



Для отладки (без отправки в Test IT) можно опустить флаг `--testit` и добавить `-s` для вывода сообщений в консоль:

```bash

pytest tests/test\_имя\_файла.py -v -s

```



\## Интеграция с Test IT

Результаты выполнения тестов автоматически отправляются в систему управления тестированием \*\*Test IT\*\*. Для корректной работы необходимо перед запуском выполнить скрипт `set\_env.ps1`, который задаёт следующие переменные окружения:

\*   `TMS\_URL` – адрес вашего экземпляра Test IT.

\*   `TMS\_PRIVATE\_TOKEN` – ваш секретный токен для доступа к API.

\*   `TMS\_PROJECT\_ID` – идентификатор проекта в Test IT.

\*   `TMS\_CONFIGURATION\_ID` – идентификатор конфигурации (например, браузер).

\*   `TMS\_ADAPTER\_MODE` – режим работы адаптера (обычно `2` для автоматического создания тестового прогона).



```powershell

.\\set\_env.ps1

pytest --testit -v

```
